A **midweight distribution** combining Debian stable base with antiX tools for an elegant, efficient
system. It features a user-friendly installer, excellent hardware detection, and helpful
configuration tools.

Its XFCE desktop with customizations provides a traditional, snappy experience that has earned it
top rankings on Distrowatch.
