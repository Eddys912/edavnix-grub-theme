A **powerful, community-developed distribution** available as Tumbleweed (rolling) and Leap
(regular release). It features the comprehensive YaST configuration tool for easy system
administration and excellent KDE Plasma integration.

Its Open Build Service and automated testing make it reliable for both desktop users and enterprise
deployments.
